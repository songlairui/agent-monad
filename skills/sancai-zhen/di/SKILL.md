# 地 · 行动路程

> 地者，脚踏之处。现状为起，终点为归，路程为证。

---

## 子技能路由

| 信号 | 执行 |
|------|------|
| 开始做、开始行动、我要做、新任务 | → [定路](#定路) |
| 下一步、行进中、继续、走到哪了 | → [行进](#行进) |
| 完成了、做完了、结束、落地 | → [落点](#落点) |

---

## 定路

> 不知终点，皆是漫游。

从天位图谱中读取相关 project/matter 作为上下文。

**提问 1 — 现状：**

> "现在，关于这件事，已有的是什么？
> （现有代码、现有认知、现有资源 — 不需要完整，说你觉得重要的）"

**提问 2 — 终态：**

> "这段行动结束时，会出现什么现在没有的东西？
> 可以是：产出物、新建立的服务、某个场景变得可能、某个问题消失。"

若用户描述过程而非终态，引导：

> "你说的是怎么走，我需要的是——走完后，世界里多了什么？"

**提问 3 — 路程边界（可选，用户可跳过）：**

> "这段行动，有没有明确不做的事？（防止边界漂移）"

生成**行动路程卡**：

```
╔══════════════════════════════════════════╗
║         行动路程 · {project/matter}      ║
╠══════════════════════════════════════════╣
║ 现状  │ {current_state}                  ║
║ 终态  │ {done_state}                     ║
║ 边界  │ {out_of_scope | 未设定}          ║
╠══════════════════════════════════════════╣
║ 关联图谱节点                             ║
║ · {node_1}                              ║
║ · {node_2}                              ║
╠══════════════════════════════════════════╣
║ 启动: {timestamp}                        ║
╚══════════════════════════════════════════╝
```

写入当前 session 状态文件 `~/.harness/session_current.json`，然后说：

> "路程已定。开始行动。"

## 行进

> 走着走着，要知道自己在哪。

加载 `~/.harness/session_current.json`。

**定期检查（每 5 个实质性步骤，或用户主动触发）：**

```
📍 行进检查
  终态: {done_state}
  
  当前位置: [你来描述，或我来判断]
  
  距终态还差: {gap_assessment}
  
  下一步: {next_action}
  
  信号: [✅ 在轨 | ⚠️ 轻微偏移 | 🔴 触发人位]
```

**偏移判断规则：**

- 当前动作对终态没有直接贡献 → ⚠️
- 超过 3 步没有推进终态 → ⚠️
- 发现前置条件缺失 → 🔴 触发人位（缺失检测）
- 范围扩展超出边界 → 🔴 触发阵眼重新定路

**触发人位格式：**
```
⚡ 阵位信号
  来源: 地
  触发: 人
  原因: {具体原因}
  携带: { session_id, missing_condition, current_step }
```

## 落点

> 终态到达，图谱更新，路程封存。

**验证终态：**

> "你现在能做到/看到 [{done_state}] 了吗？"

- 是 → 完成流程
- 部分 → 记录差距，转为新的播种种子进入天位
- 否 → 重新进入行进，或触发人位

**完成流程：**

1. 将完成的 action 节点标记为 `resolved`
2. 将路程中涌现的 insights 写回天位图谱
3. 将行动路程卡归档到 `~/.harness/sessions/`
4. 清空 `session_current.json`
5. 输出落点摘要：

```
✅ 落点
  终态已达: {done_state}
  
  图谱更新:
  · [resolved] {action_node}
  · [new insight] {insight}  → 已写入天位
  
  路程归档: ~/.harness/sessions/{session_id}.json
  
  下一步？[播种新想法] [开始新行动] [查看全景]
```
