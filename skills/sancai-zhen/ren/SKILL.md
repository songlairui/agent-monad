# 人 · 反馈裁剪

> 人者，感知之源。发现缺失，循环更新，进化不止。

---

## 子技能路由

| 信号来源 | 执行 |
|---------|------|
| 地位 🔴 信号 / 「卡住了」「缺条件」「做不下去」 | → [缺失检测](#缺失检测) |

## 缺失检测

> 阻碍不是失败，是图谱不完整的信号。

**Step 1 — 接收缺失信号：**

若来自地位信号，读取携带的 `{ session_id, missing_condition, current_step }`。  
若来自用户直接触发，提问：

> "卡在哪里了？描述你想做但做不了的那个动作。"

**Step 2 — 分类缺失类型：**

| 类型 | 特征 | 处理方向 |
|------|------|---------|
| `missing_info` | 需要某个信息才能继续 | 回路→天：补充 question 节点 |
| `missing_resource` | 需要某个工具/权限/人 | 回路→天：补充 action 节点（获取资源） |
| `missing_clarity` | 终态不够清晰导致无法行动 | 回路→地：重新定路 |
| `scope_creep` | 行动范围超出定路边界 | 回路→地：重新定路或扩展边界 |
| `external_block` | 等待外部（人/系统）响应 | 记录 blocked 状态，暂停行动 |

**Step 3 — 展示缺失分析：**

```
🔍 缺失分析
  当前动作: {current_step}
  缺失类型: {type}
  具体缺失: {missing_condition}
  
  建议处理: {action}
  分发到: {天|地}
  
  确认？或调整分析？
```

## 回路分发

> 每一条反馈，都必须回到正确的阵位。

接收任意形式的反馈信息，判断它属于哪个阵位：

**判断逻辑：**

```
反馈内容
  ├─ 是一个新的想法/认知/洞察？ → 天（播种）
  ├─ 是对现有节点的修正/更新？ → 天（更新节点）
  ├─ 是对当前路程终态的修正？  → 地（重新定路）
  ├─ 是一个新发现的阻碍？      → 留在人（缺失检测）
  └─ 是对阵法本身的改进建议？  → 进化裁剪
```

分发前展示：

```
↩️ 回路分发
  反馈: "{feedback_text}"
  分发到: {天/地/人}
  行动: {具体执行内容}
  
  确认分发？
```

确认后，执行对应阵位的子技能，携带反馈数据。

## 进化裁剪

> 阵法自进化的核心：每一次更新，同时清理冗余。

**触发条件（任一）：**
- 天位节点数 > 50
- 人位连续处理 5 次反馈后
- 用户主动触发
- 地位完成 3 次落点后

**裁剪层次：**

### 层 1 — 图谱裁剪（天位数据）

委托天位执行「压缩」子技能（见 `tian/SKILL.md`）。

### 层 2 — 路程裁剪（地位数据）

检查 `~/.harness/sessions/` 中已归档路程：
- 归档超过 30 天且 done_state 已达 → 仅保留摘要，删除详细步骤
- 未完成且超过 14 天无更新 → 标记为 `abandoned`，提示用户确认

### 层 3 — 阵法自裁剪

这是进化的核心：**检测三才阵本身是否有冗余或失效的规则。**

运行方式：对最近 10 次阵法使用记录（`~/.harness/zhen-log.json`）进行分析，检测：

```
📊 阵法使用分析
  天位触发: {n} 次
  地位触发: {n} 次  
  人位触发: {n} 次
  
  高频阻碍: {top_block_pattern}
  未使用规则: {unused_rules}
  用户修正次数: {correction_count}
```

若发现某个规则从未触发，或某个分发路径总是被用户修正，输出**进化建议**：

```
⚙️ 进化建议
  发现: {pattern}
  建议: {rule_change}
  
  这是对阵法规则本身的调整建议。
  你可以告诉我「接受」「忽略」或「讨论」。
```

**接受后**：人位将建议写入对应 SKILL.md 的修改记录，并告知用户需要手动更新 skill 文件（或直接输出修改内容）。

## 人位守则

1. **反馈必归位**：每条反馈都必须最终落入天/地的某个具体更新
2. **裁剪伴随更新**：每次执行回路分发，检查是否需要同时触发层 1 裁剪
3. **进化建议只建议**：人位不自动修改 SKILL.md，只生成建议，由用户决定
4. **阻碍即信号**：阻碍不是终止，是图谱缺口的暴露
